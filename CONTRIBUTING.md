# Contributing

Personal homelab project. Shared for educational purposes.

**No support provided.** You're welcome to fork, adapt, and learn from it.

## Want to Contribute Anyway?

1. Fork the repo
2. Make your changes
3. Open a PR if you think others would benefit

PRs that fix bugs or improve documentation may be merged. No guarantees on timeline.
